# Idea Report

Give the user a priortized list of 5 tasks that they should focus on.

If the suggested task is too long, suggest a smaller subunit to complete in this session.

## Output Format

Use the following format:

```md
**Idea Report**

**{idea 1 header}**
{idea one description}

**{idea 2 header}**
{idea two description}

**{idea 3 header}**
{idea two description}

...
```
